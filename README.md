# Bank IFSC Search
![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)

IFSC_API has been developed on Django (Python 3.7.3)

## Installation

Clone the repository and run the following command

```bash
py manage.py runserver
```
